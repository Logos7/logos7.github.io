Yantra

A modular platform for designing FPGA systems from blocks — from simple circuits to processors, graphics, audio and experimental architectures.

What is it?

Yantra is an attempt to build an environment in which hardware systems can be assembled from modules, parameterized, connected and deployed to different FPGA boards without rebuilding the entire process manually from scratch.

The project combines graphical editors, code generation, target support, build tools and custom description languages. HDL remains important, but it does not have to be the only level of work.

Direction

  • graphical composition of systems from blocks,
  • parameterized modules and multiple FPGA targets,
  • the Mantra language compiled to Sutra,
  • support for processors, graphics, audio and specialized coprocessors,
  • demonstration projects showing the full path from an idea to working hardware.
Idea: FPGA not as a collection of separate experiments, but as a coherent ecosystem for building new machines.
