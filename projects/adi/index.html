Adi

An experimental FPGA systems laboratory where processors, synthesizers, graphics circuits and hardware interfaces are built.

GitHub repository → All projects

What is it?

Adi is a collection of working hardware experiments developed directly on FPGA boards. It is used to test ideas, build small machines and learn hardware from the inside.

Areas of experimentation

  • programmable processors and custom instructions,
  • sound synthesis, DDS, ADSR and hardware mixers,
  • graphics generated directly in FPGA,
  • SPI displays, UART and PMOD modules,
  • ray tracing and specialized computing circuits.
Project role: rapid experimentation and preserving working circuits as a foundation for future systems.
